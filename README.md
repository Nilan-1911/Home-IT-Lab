# 🏠 Home IT Lab

A self-hosted Home IT Lab built to simulate a real-world enterprise IT environment using Debian Linux, Docker, VirtualBox, and open-source infrastructure tools. This project focuses on developing practical Linux system administration, IT support, infrastructure management, networking, monitoring, documentation, and troubleshooting skills through hands-on implementation.

The lab is designed to replicate common responsibilities performed by Help Desk Engineers, System Administrators, Linux Administrators, and Infrastructure Engineers.

---

# Project Goals

* Build and manage a production-like Linux server environment.
* Gain hands-on experience with Linux system administration.
* Learn Docker container deployment and service management.
* Practice remote server administration using SSH.
* Implement infrastructure monitoring and documentation.
* Simulate enterprise IT operations and Help Desk workflows.
* Develop troubleshooting and operational documentation.
* Build a portfolio project demonstrating real infrastructure skills.
